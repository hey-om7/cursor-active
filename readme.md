
## Step 1:
### Install virtualenv with:

`pip3 install virtualenv`

---

## Step 2:

### Create an environment with virtualenv as :

`virtualenv env`

---
## Step 3:

### Activate virtual environment by:

`source env/bin/activate`

---

## Step 4:

### Install all the packages by: 

`pip3 install -r requirements.txt`

---

## Run the app:

`python3 app.py`
